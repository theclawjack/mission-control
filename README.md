# 🚀 Jashboard

AI Agent Dashboard for OpenClaw — built by Jack (Opus 4.6) and the team.

## Modules
- **📋 Tasks Board** — Kanban-style task management with team assignees
- **🧠 Memory** — Browse and search all agent memory files
- **📅 Calendar** — Visual calendar for scheduled tasks and events
- **👥 Teams** — Agent roster with roles, models, and status

## Tech Stack
- Next.js 14 (App Router)
- SQLite (better-sqlite3)
- Tailwind CSS (dark theme)
- Cookie-based password auth

## Setup
```bash
npm install
npm run build
PORT=3100 npm start
```
